# 开放生态 PHP SDK

## 语言说明
语言: php >= 5.5

## 安装依赖
install:
  - "composer install"

## 运行demo
1.更改access_token等参数值
2.在demo目录中执行`php -f "XpanMain.php"`

## 在项目中使用

1.在项目中引入自动加载文件 `require_once(__DIR__ . '/../vendor/autoload.php');`
2.参照demo中的示例在项目中编写代码
